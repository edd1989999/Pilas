# Diseño de Pilas

## Introducción
Este repositorio contiene los archivos de diseño de pilas, incluyendo cálculos geotécnicos y estructurales, informes en Word y PDF, y diagramas de flujo.

## Objetivos
- Presentar el diseño completo de pilas según métodos reconocidos.  
- Documentar parámetros de entrada y resultados.  
- Facilitar la descarga de los archivos para su uso en proyectos.

## Archivos incluidos
- `Diseño_Pilas.xlsx` → Cálculos completos.  
- `Informe_Diseño_Pilas.docx` → Informe detallado.  
- `Informe_Diseño_Pilas.pdf` → PDF listo para entregar.  
- `Diagramas_Pilas.zip` → Diagramas y gráficos.
